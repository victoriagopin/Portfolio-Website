  • Movie Magic is a web application developed using Express.js and Handlebars for rendering dynamic pages, catering to movie enthusiasts. It integrates MongoDB as the primary database for managing persistent data. The application utilizes Express Router for route handling, ensuring clean and modular code structure. To ensure security, the application employs bcrypt for hashing user passwords before storing them in the database.
  • Provides users with the ability to browse available movies and their descriptions.
  • Guest users can:
    • Browse the list of uploaded movies
    • View movie descriptions
    • Access the search page to find specific movies
  • Registered users have extended functionality, allowing them to:
    • Create new movies on the platform
    • Add cast members to created movies
